# android-starter

Android Starter is An Android Project loaded with most needed features to start an app.

The project showcasing the following:
  - AndroidX
  - Generics
  - Permissions requesting
  - SharedPreferences
  - Dialogs
  - Splash Screen
  - Onboarding Screen with ViewPager2
  - Login with email and password
  - Login with social media auth providers
  - MVP architectural pattern
  - Startegy design pattern to get user's data from differnet social media auth providers
  - Network calls with Retrofit
  - Navigation Component
  
Just follow the included TODOs to get the app up and runing :)
